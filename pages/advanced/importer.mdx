# Importer

JSON Importer allows us to create schema in a rapid way, by avoiding designing field by field. 

To import schema via importer. 

1. Go to schema tab
2. Click on Import
3. Enter a valid JSON 
4. Enter a valid root object name
5. Click on Import entities

Note:

JSON Importer cannot diferentiate between `String` and `Enum` type, will be considered as `String` by default.
