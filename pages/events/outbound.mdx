# Outbound Event

What is an outbound event ?

If a microservice wants to send / publish event to another service, outbound event can be used to send data. 

Steps to create an outbound event.

1. Go to events tab
2. Click on add and select outbound event category, enter a name and confirm on plus button
3. Choose the channel, more explanation on [channels](./channels)
4. Choose the schema of outbound event
5. All fields are selected by default u can choose to modify this
