# Channels

Channel points to one or more topic in radical. A topic can contain one or more event types. 
Channels are common for both outbound event and inbound event. 

Steps to create channel

1. Go to events tab
2. Click on add and select channels category, enter a name and confirm on plus button
3. Topic name where messages has to be consumed or produced. 
4. A channel can be associated with only one [binder](./binders)
5. A channel can describe whether it has one type of events or multiple
